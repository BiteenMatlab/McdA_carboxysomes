# McdA_Carboxysomes

Data analysis scripts used in Azaldegui et al 2026 "The kinetics and mobility of McdA drive carboxysome distribution in Halothiobacillus neapolitanus"

Scripts are organized into subdirectories corresponding to the type of analysis performed:
- biochem_analysis
- bulk_imaging_analysis
- copy_number_analysis
- spt_diffusion_analysis
- spt_dissociation_rate
